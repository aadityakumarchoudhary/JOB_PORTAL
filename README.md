
# MERN Stack Job Portal

## Frontend
- React + Vite
- Redux Toolkit
- Tailwind CSS
- React Router

## Backend
- Node.js
- Express
- MongoDB

## Run Frontend
cd frontend
npm install
npm run dev

## Run Backend
cd backend
npm install
node index.js

## Deployment

### Frontend
Deploy on Vercel

### Backend
Deploy on Render

## CI/CD
Connect GitHub repo with Vercel + Render for automatic deployment.
